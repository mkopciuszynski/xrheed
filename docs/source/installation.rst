Installation
============

Install xRHEED using pip:

.. code-block:: bash

    pip install xrheed

Or install the latest version from source:

.. code-block:: bash

    git clone https://github.com/mkopciuszynski/xrheed.git
    cd xrheed
    pip install .

Dependencies
------------

- Python 3.12+
- xarray
- numpy
- matplotlib

Check ``requirements.txt`` for the full list.
